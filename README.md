# rainyproxy
(plan) customizable local proxy server in Rust. It's work in progress.

    USAGE:
            rainy [FLAGS] [OPTIONS]
    
    FLAGS:
            --help       Prints help information
        -V, --version    Prints version information
    
    OPTIONS:
        -h, --host <HOST>
            --loglevel <LOG_LEVEL>
        -p, --port <PORT>
